Dichotomous Keys - Exam Questions

Exam-style questions on using and constructing dichotomous keys for identification

* Exam-Style Questions

Question 1 [3 marks]

(a) Define the term 'dichotomous key'. [2 marks]

(b) State one use of a dichotomous key. [1 mark]

* CLICK TO REVEAL ANSWER
  • (a) A series of paired statements / two-choice questions (1 mark)
  • (a) Based on observable features of organisms (1 mark)
  • (b) To identify organisms / species / unknown specimens (1 mark)
Question 2 [4 marks]

Use the dichotomous key below to identify organisms A and B.

Step Statement A Statement B
1 Has a backbone ... go to 2 Has no backbone ... go to 4
2 Has feathers ... Bird Has no feathers ... go to 3
3 Has scales ... Fish Has fur or hair ... Mammal
4 Has six legs ... Insect Has eight legs ... Spider

Organism A: Has no backbone and has six legs.

Organism B: Has a backbone, no feathers, and has fur.

* CLICK TO REVEAL ANSWER

Organism A:

  • Step 1: No backbone, go to 4 (1 mark)
  • Step 4: Six legs = Insect (1 mark)

Organism B:

  • Step 1: Has backbone, go to 2; Step 2: No feathers, go to 3 (1 mark)
  • Step 3: Has fur = Mammal (1 mark)
Question 3 [4 marks]

State two features that would be suitable for use in a dichotomous key, and two features that would not be suitable. Explain your choices.

* CLICK TO REVEAL ANSWER

Suitable features (any two):

  • Number of legs - can be counted precisely / does not vary within a species (1 mark)
  • Presence or absence of wings - clearly observable / either present or not (1 mark)
  • Type of body covering (scales, feathers, fur) - constant within a species (1 mark)

Unsuitable features (any two):

  • Colour - can vary between individuals of the same species (1 mark)
  • Size - changes with age / varies between individuals (1 mark)
  • Behaviour - difficult to observe / not always visible (1 mark)
  • Habitat - organisms can move / not a physical feature (1 mark)
Question 4 [5 marks]

The table below shows features of four invertebrates.

Organism Legs Wings Body segments
Housefly 6 Yes 3
Ant 6 No 3
Spider 8 No 2
Woodlouse 14 No 7

Construct a dichotomous key that could be used to identify these four invertebrates.

* CLICK TO REVEAL ANSWER

Example dichotomous key:

1a Has 6 legs ... go to 2
1b Has more than 6 legs ... go to 3
2a Has wings ... Housefly
2b Has no wings ... Ant
3a Has 8 legs ... Spider
3b Has 14 legs ... Woodlouse

Award marks for:

  • Correct use of paired statements (1 mark)
  • Using observable features from the table (1 mark)
  • Logical structure that leads to correct identification (1 mark)
  • Clear numbering system (1 mark)
  • All four organisms can be correctly identified (1 mark)
Question 5 [3 marks]

Explain why colour is usually not a good feature to use in a dichotomous key for identifying animals.

* CLICK TO REVEAL ANSWER
  • Colour can vary between individuals of the same species (1 mark)
  • Colour can change with age / season / environment (1 mark)
  • Colour perception can be subjective / different people may describe colours differently (1 mark)
  • Many different species can have the same colour (1 mark) (any three)
Question 6 [4 marks]

A student is trying to create a dichotomous key for five species of tree leaves. They write the following first step:

"1a. Leaf is large ... go to 2"

"1b. Leaf is not large ... go to 3"

(a) Explain why this is not a good statement for a dichotomous key. [2 marks]

(b) Suggest a better alternative statement the student could use. [2 marks]

* CLICK TO REVEAL ANSWER
  • (a) "Large" is vague / subjective / not precisely defined (1 mark)
  • (a) Different people may interpret "large" differently / leaf size varies within species (1 mark)
  • (b) Use a measurable criterion, e.g., "Leaf is longer than 10 cm" or "Leaf is wider than 5 cm" (1 mark)
  • (b) Or use a different observable feature that is more definite, e.g., "Leaf has pointed edges" vs "Leaf has rounded edges" (1 mark)
Question 7 [6 marks]

A scientist discovers six new species of beetle in a rainforest. The table shows their features.

Beetle Antennae length Wing covers Leg spines
Species A Long Striped Present
Species B Long Striped Absent
Species C Long Plain Present
Species D Long Plain Absent
Species E Short Striped Present
Species F Short Plain Absent

Construct a complete dichotomous key to identify all six beetle species.

* CLICK TO REVEAL ANSWER

Example dichotomous key:

1a Antennae are long ... go to 2
1b Antennae are short ... go to 5
2a Wing covers are striped ... go to 3
2b Wing covers are plain ... go to 4
3a Leg spines present ... Species A
3b Leg spines absent ... Species B
4a Leg spines present ... Species C
4b Leg spines absent ... Species D
5a Wing covers are striped ... Species E
5b Wing covers are plain ... Species F

Award marks for:

  • Correct paired statements throughout (2 marks)
  • Logical use of features to distinguish species (2 marks)
  • All six species correctly identifiable (2 marks)
