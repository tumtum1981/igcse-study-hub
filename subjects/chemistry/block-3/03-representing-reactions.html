C3.3.1

Representing Reactions

Summary

Chemical reactions can be represented using word equations and symbol equations. Word equations show the names of reactants and products. Symbol equations use chemical formulae and show the exact atoms involved.

Word Equations

Word equations show the names of the substances involved in a reaction:

  • Reactants are on the left side (substances that react)
  • Products are on the right side (substances that are formed)
  • An arrow (→) shows the direction of the reaction
  • + signs separate multiple reactants or products
FORMAT

Reactant 1 + Reactant 2 → Product 1 + Product 2

Examples:

  • magnesium + oxygen → magnesium oxide
  • calcium carbonate → calcium oxide + carbon dioxide
  • hydrochloric acid + sodium hydroxide → sodium chloride + water

Symbol Equations

Symbol equations use chemical formulae instead of names:

  • Show the exact atoms and molecules involved
  • Must be balanced (same number of each atom on both sides)
  • Can include state symbols

Examples:

  • 2Mg + O2 → 2MgO
  • CaCO3 → CaO + CO2
  • HCl + NaOH → NaCl + H2O

State Symbols

State symbols show the physical state of each substance:

Symbol State Example
(s) Solid NaCl(s)
(l) Liquid H2O(l)
(g) Gas CO2(g)
(aq) Aqueous (dissolved in water) NaCl(aq)

Example with state symbols:

CaCO3(s) → CaO(s) + CO2(g)

Chemical Formulae

To write symbol equations, you need to know the formulae of common substances:

Substance Formula
Water H2O
Carbon dioxide CO2
Oxygen O2
Hydrogen H2
Hydrochloric acid HCl
Sodium hydroxide NaOH
Sodium chloride NaCl
Magnesium oxide MgO
Calcium carbonate CaCO3

Converting Word Equations to Symbol Equations

  1. Write out the word equation
  2. Replace each name with its chemical formula
  3. Balance the equation (next topic)
  4. Add state symbols if required

Example:

Word: magnesium + oxygen → magnesium oxide

Symbol: Mg + O2 → MgO (unbalanced)

Balanced: 2Mg + O2 → 2MgO

Key Points to Remember

  • Reactants are on the left, products on the right
  • The arrow means "produces" or "forms"
  • State symbols show solid (s), liquid (l), gas (g), or aqueous (aq)
  • Symbol equations must be balanced
  • Learn common chemical formulae
