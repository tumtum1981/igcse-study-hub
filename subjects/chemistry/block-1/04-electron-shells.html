C1.4.1

Electron Shells

Summary

Electrons are arranged in shells (also called energy levels) around the nucleus. Each shell can hold a maximum number of electrons, following the 2.8.8 rule. The electron configuration describes how electrons are distributed across these shells.

Electron Shell Capacity

Shell Number Name Maximum Electrons Distance from Nucleus
1st shell K shell 2 Closest
2nd shell L shell 8 Further
3rd shell M shell 8 (for IGCSE) Furthest

The 2.8.8 Rule

Electrons fill shells from the inside out:

First shell (max 2) - Second shell (max 8) - Third shell (max 8)

  • Electrons always fill the innermost shell first
  • Only when a shell is full do electrons go into the next shell
  • The outer shell electrons are called valence electrons

Electron Configuration and the Periodic Table

  • Group number = Number of electrons in the outer shell
  • Period number = Number of occupied electron shells
  • Elements in the same group have the same number of outer electrons
  • This is why elements in the same group have similar chemical properties
  • Noble gases (Group 0/18) have full outer shells - they are very stable and unreactive

Key Points to Remember

  • Electrons fill shells closest to the nucleus first
  • The 2.8.8 rule tells us maximum electrons per shell
  • Outer shell electrons determine chemical properties
  • A full outer shell makes an atom stable (like noble gases)
  • Atoms react to gain, lose, or share electrons to get a full outer shell

Question 1 [2 marks]

State the maximum number of electrons that can occupy the first and second electron shells.

Question 2 [3 marks]

An element has an electron configuration of 2.8.6. Identify the element and state which group and period it is in.

* CLICK TO REVEAL ANSWER
  • The element is sulfur (S), with atomic number 16 (1 mark)
  • Group 6 (6 electrons in outer shell) (1 mark)
  • Period 3 (3 occupied shells) (1 mark)
Question 3 [2 marks]

Explain why elements in the same group of the periodic table have similar chemical properties.

* CLICK TO REVEAL ANSWER
  • They have the same number of electrons in their outer shell (1 mark)
  • Chemical properties depend on the number of outer shell electrons (1 mark)
Question 4 [2 marks]

Explain why noble gases are unreactive.

* CLICK TO REVEAL ANSWER
  • Noble gases have a full outer shell of electrons (1 mark)
  • They do not need to gain, lose, or share electrons / they are already stable (1 mark)
