Atoms, Elements and Compounds

Summary

All matter is made up of tiny particles called atoms. An element is a substance made of only one type of atom. A compound is a substance made of two or more different types of atoms chemically joined together.

Key Definitions

Term Definition
Atom The smallest particle of an element that can exist
Element A substance made of only one type of atom
Compound A substance made of two or more different elements chemically bonded together
Molecule Two or more atoms chemically bonded together
Mixture Two or more substances not chemically bonded together

Elements vs Compounds vs Mixtures

Type Description Example
Element Only one type of atom Oxygen (O2), Iron (Fe), Gold (Au)
Compound Different atoms chemically bonded Water (H2O), Carbon dioxide (CO2)
Mixture Substances not chemically bonded Air, salt water, alloys

The Periodic Table

  • Elements are arranged by atomic number (number of protons)
  • Rows are called periods
  • Columns are called groups
  • Elements in the same group have similar properties
  • Metals are on the left, non-metals on the right

Chemical Formulae

Chemical formulae show which elements are in a compound and how many atoms of each:

  • H2O = 2 hydrogen atoms + 1 oxygen atom
  • CO2 = 1 carbon atom + 2 oxygen atoms
  • NaCl = 1 sodium atom + 1 chlorine atom

Key Points to Remember

  • Atoms are the smallest particle of an element
  • Elements contain only one type of atom
  • Compounds have atoms chemically bonded
  • Mixtures can be separated by physical methods
  • The periodic table organises elements by atomic number
