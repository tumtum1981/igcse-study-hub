SB1.4.1

Dichotomous Keys

Summary

A dichotomous key is a tool used to identify and classify organisms. At each step, you are given two choices based on observable features. By answering a series of questions, you can identify an unknown organism. Scientists and field biologists use dichotomous keys to classify organisms into their correct groups.

Key Definitions

Term Definition
Dichotomous key A series of paired statements used to identify organisms by their features
Classification The process of sorting organisms into groups based on shared features
Observable features Characteristics that can be seen, measured, or detected

Using a Key to Classify Organisms

Some organisms look like they may belong to several groups. For example, a penguin has wings but cannot fly - is it a bird or a mammal? A classification key helps us decide:

Step Question Yes No
1 Does it have feathers? It's a bird Go to 2
2 Does it have fur and give birth to live young? It's a mammal Go to 3
3 Does it have gills and scales? It's a fish Go to 4
4 Does it have dry scales and lay eggs on land? It's a reptile It's an amphibian
  • Penguin: Has feathers → It's a bird (even though it can't fly)
  • Bat: No feathers → Has fur and live young → It's a mammal (even though it flies)
  • Whale: No feathers → Has fur (some) and live young → It's a mammal (even though it lives in water)

Building a Classification Key

When building a key, you need to choose questions with specific answers that lead to only one route for any animal. Consider classifying an armadillo, ostrich, dolphin, and turtle:

Things to consider:

  • Which features help you tell the animals apart?
  • Which questions would help somebody decide how to classify the animals?
  • Which features suggest an animal should be in a particular group?

Example Key

Step Question Yes No
1 Is it warm-blooded? Go to 2 Turtle (reptile)
2 Does it give birth to live young? Go to 3 Ostrich (bird)
3 Does it have a hard shell? Armadillo (mammal) Dolphin (mammal)

Correct classifications: Armadillo = mammal, Ostrich = bird, Dolphin = mammal, Turtle = reptile

Practical: Crab Classification

Dichotomous keys can also be used to identify species within a group. Crabs are classified as crustaceans (along with prawns, shrimps and lobsters). Crustaceans, together with insects, arachnids and myriapods, belong to the group called arthropods.

To build a key for different crab species, you would look for visible features such as:

  • Size and shape of the carapace (shell)
  • Size of the chelipeds (claws) - are they equal or is one larger?
  • Leg shape - walking legs vs flattened paddle-like legs
  • Body markings, spines, or hair

Importance of Classification

  • Helps us organise and make sense of the biodiversity on Earth
  • Allows us to monitor and track diversity and conservation efforts
  • Provides a standard system for identifying and naming organisms

Key Points to Remember

  • Dichotomous keys use pairs of contrasting questions to identify organisms
  • Each step leads to either another question or an identification
  • Use observable features like body covering, number of legs, how they reproduce
  • Keys can be written as numbered steps or as branching diagrams
  • The same set of organisms can be classified using different valid keys
