SB1.4.1

Dichotomous Keys

Summary

A dichotomous key is a tool used to identify organisms. The word "dichotomous" means "dividing into two parts" - at each step, you are given two choices based on observable features. By answering a series of yes/no or either/or questions, you can identify an unknown organism. Scientists and field biologists use dichotomous keys to identify species accurately.

Key Definitions

Term Definition
Dichotomous key A series of paired statements used to identify organisms by their features
Identification The process of determining which species an organism belongs to
Observable features Characteristics that can be seen, measured, or detected
Paired statements Two contrasting descriptions, only one of which matches the organism

How to Use a Dichotomous Key

Follow these steps to identify an organism using a dichotomous key:

  1. Start at question 1 (or the first pair of statements)
  2. Read both options and choose the one that matches your organism
  3. Follow the instruction - either go to another question or identify the organism
  4. Continue until you reach a species identification

Example Dichotomous Key: Invertebrates

Step Statement A Statement B
1 Has legs ... go to 2 Has no legs ... go to 4
2 Has 6 legs ... go to 3 Has 8 legs ... Spider
3 Has wings ... Fly Has no wings ... Ant
4 Has a shell ... Snail Has no shell ... Worm

Creating a Dichotomous Key

When creating your own dichotomous key, follow these guidelines:

  • Use observable features: Choose characteristics that can be easily seen or measured
  • Make statements opposite: Each pair should be mutually exclusive (only one can be true)
  • Use positive statements: Say "has wings" rather than "does not have fins"
  • Be specific: Avoid vague terms like "large" or "small" - use measurable criteria
  • Start with major differences: Begin with the most obvious distinguishing features
  • Number your steps: Use clear numbering to guide users through the key

Types of Features to Use

Good Features Features to Avoid
Number of legs (0, 6, 8) Colour (can vary within species)
Presence/absence of wings Size (changes with age)
Body segments (yes/no) Behaviour (hard to observe)
Type of body covering (scales, feathers, fur) Habitat (organisms can move)
Leaf shape or arrangement Subjective descriptions (pretty, ugly)

Key Points to Remember

  • Dichotomous keys use pairs of contrasting statements
  • Each step leads to either another question or an identification
  • Use observable, measurable features that do not vary
  • Make each pair of statements mutually exclusive
  • Keys can be written as numbered steps or as branching diagrams

Question 2 [2 marks]

State two features of an organism that would be suitable for use in a dichotomous key, and explain why.

* CLICK TO REVEAL ANSWER
  • Number of legs - because it is a definite number that can be counted/does not vary (1 mark)
  • Presence or absence of wings - because it is clearly observable/either present or not (1 mark)
  • Type of body covering - because it is constant within a species (1 mark) (any two with valid explanations)
Question 3 [3 marks]

Use the dichotomous key below to identify an organism that has no legs and no shell.

1a. Has legs ... go to 2 | 1b. No legs ... go to 3
2a. Has 6 legs ... Beetle | 2b. Has 8 legs ... Spider
3a. Has a shell ... Snail | 3b. No shell ... Earthworm

* CLICK TO REVEAL ANSWER
  • Start at 1 - no legs, so go to 3 (1 mark)
  • At 3 - no shell, so choose 3b (1 mark)
  • The organism is an Earthworm (1 mark)
Question 4 [3 marks]

Explain why colour is usually not a good feature to use in a dichotomous key.

* CLICK TO REVEAL ANSWER
  • Colour can vary between individuals of the same species (1 mark)
  • Colour can change with age/season/environment (1 mark)
  • Colour perception can be subjective/different people may see colours differently (1 mark)
