Kingdom Features in Detail

Supplements: SB1.1.1 Features of Organisms

Kingdom Key Features Examples
Animals Multicellular, no cell wall, heterotrophic (feed on other organisms), can move Humans, fish, insects, birds
Plants Multicellular, cellulose cell wall, autotrophic (make own food by photosynthesis), contain chloroplasts Trees, grasses, ferns, mosses
Fungi Mostly multicellular, chitin cell wall, saprophytic nutrition (feed on dead matter), no chloroplasts Mushrooms, yeast, moulds
Protoctists Mostly unicellular, eukaryotic (have a nucleus), varied nutrition methods Amoeba, Paramecium, algae
Prokaryotes Unicellular, no true nucleus (DNA free in cytoplasm), cell wall (not cellulose) Bacteria, cyanobacteria

Features Used for Classification

Scientists use various observable and structural features to classify organisms:

  • Cell type: Prokaryotic (no nucleus) or eukaryotic (has nucleus)
  • Cell structure: Presence of cell wall, type of cell wall material
  • Number of cells: Unicellular or multicellular
  • Nutrition: Autotrophic (makes own food) or heterotrophic (feeds on others)
  • Body structure: Presence of tissues, organs, body symmetry
  • Reproduction: Sexual or asexual, type of reproductive structures

Why Classification is Important

  • Helps scientists communicate about organisms using standard names
  • Shows evolutionary relationships between organisms
  • Helps identify new species and understand biodiversity
  • Organises biological knowledge in a logical way
  • Assists in conservation efforts by understanding species relationships

Advantages of the Binomial System

Supplements: SB1.2.1 Species and Binomial Naming

  • Universal: The same scientific name is used worldwide, avoiding confusion from different common names
  • Unique: Each species has only one scientific name
  • Shows relationships: Organisms in the same genus share the first part of their name (e.g., Panthera leo and Panthera tigris are closely related)
  • Precise: Avoids ambiguity - common names can refer to different organisms in different regions
  • Stable: Scientific names rarely change, while common names vary

Classification Hierarchy Mnemonic

"King Philip Came Over For Good Soup" - Kingdom, Phylum, Class, Order, Family, Genus, Species

Evolutionary Trees - Formal Definitions and Reading Guide

Supplements: SB1.3.1 Evolutionary Trees

Key Definitions

Term Definition
Phylogenetic tree A diagram showing the evolutionary relationships between organisms
Node A branching point on a tree, representing a common ancestor
Divergence When one species splits into two or more separate species

Reading Evolutionary Trees

Feature What It Represents
Root (base) The oldest common ancestor of all organisms on the tree
Branches Lines of evolutionary descent from ancestors to descendants
Nodes (branching points) Points where species diverged; represent common ancestors
Tips (ends of branches) Living species or groups (or extinct species at dead ends)
Branch length May represent time or amount of genetic change (depends on tree type)

Non-DNA Evidence for Evolutionary Trees

Type of Evidence Description
Physical features (morphology) Comparing body structures, skeletal features, and anatomy
Protein sequences Comparing amino acid sequences in proteins like haemoglobin
Fossil record Examining preserved remains of ancient organisms
Embryo development Comparing how embryos develop in different species

Dichotomous Keys - Examples and Guidelines

Supplements: SB1.4.1 Dichotomous Keys

Example Dichotomous Key: Invertebrates

Step Statement A Statement B
1 Has legs ... go to 2 Has no legs ... go to 4
2 Has 6 legs ... go to 3 Has 8 legs ... Spider
3 Has wings ... Fly Has no wings ... Ant
4 Has a shell ... Snail Has no shell ... Worm

Creating a Dichotomous Key

When creating your own dichotomous key, follow these guidelines:

  • Use observable features: Choose characteristics that can be easily seen or measured
  • Make statements opposite: Each pair should be mutually exclusive (only one can be true)
  • Use positive statements: Say "has wings" rather than "does not have fins"
  • Be specific: Avoid vague terms like "large" or "small" - use measurable criteria
  • Start with major differences: Begin with the most obvious distinguishing features
  • Number your steps: Use clear numbering to guide users through the key

Types of Features to Use

Good Features Features to Avoid
Number of legs (0, 6, 8) Colour (can vary within species)
Presence/absence of wings Size (changes with age)
Body segments (yes/no) Behaviour (hard to observe)
Type of body covering (scales, feathers, fur) Habitat (organisms can move)
Leaf shape or arrangement Subjective descriptions (pretty, ugly)

Plant Classification - Extended Detail

Supplements: SB1.5.1 Plant Classification

Conifers (Gymnosperms)

Conifers are gymnosperms - plants that produce seeds not enclosed in a fruit. Key features:

  • Have vascular tissue (xylem and phloem)
  • Produce seeds in cones (not flowers)
  • Needle-like or scale-like leaves
  • Most are evergreen (keep leaves year-round)
  • No flowers or fruits
  • Examples: pine, spruce, fir trees

Mosses and Ferns - Expanded Detail

Feature Mosses (Bryophytes) Ferns (Pteridophytes)
True roots No (have rhizoids) Yes
True stems and leaves No Yes
Vascular tissue No Yes (xylem and phloem)
Reproduction Spores; need water for male gametes to swim Spores on underside of leaves (sporangia)
Habitat restriction Damp habitats (no water transport) Can grow in drier areas (have vascular tissue)

Flowering vs Non-Flowering Plants

Feature Flowering Plants Non-Flowering Plants
Flowers Present - for reproduction Absent
Seeds Enclosed in fruit Exposed in cones or absent (spores instead)
Reproduction Seeds produced after pollination Spores or naked seeds
Examples Apple tree, tulip, wheat Fern, moss, pine tree

Vascular Tissue

Vascular tissue is the transport system in plants:

  • Xylem: Transports water and minerals from roots to leaves
  • Phloem: Transports sugars (from photosynthesis) around the plant

Vascular Bundle Arrangement

Feature Monocots Dicots
Vascular bundles in stem Scattered throughout stem Arranged in a ring
Root system Fibrous roots (many thin roots) Taproot (one main root)
