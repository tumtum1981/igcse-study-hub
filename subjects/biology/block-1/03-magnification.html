B1.3.1

Magnification

Summary

Magnification is how many times larger an image is compared to the actual size of the specimen. We use microscopes to view cells and other small structures that are too small to see with the naked eye.

The Magnification Formula

Magnification = Image Size / Actual Size
M = I / A

This formula can be rearranged to find any of the three values:

  • Image size = Magnification x Actual size (I = M x A)
  • Actual size = Image size / Magnification (A = I / M)
  • Magnification = Image size / Actual size (M = I / A)

Remember the MAI triangle: I on top, M and A on the bottom.

Units and Conversions

When using the magnification formula, units must be the same. Common conversions:

Conversion How to convert
mm to um (micrometres) Multiply by 1000
um to mm Divide by 1000
cm to mm Multiply by 10
mm to cm Divide by 10

1 mm = 1000 um (micrometres)

Types of Microscopes

Type Maximum Magnification Features
Light microscope x1500 Uses light, can view living specimens, lower resolution
Electron microscope x2,000,000 Uses electrons, much higher resolution, specimens must be dead

Calculating Total Magnification

Total Magnification = Eyepiece lens x Objective lens
e.g., x10 eyepiece x x40 objective = x400 total

Key Points to Remember

  • Always check units are the same before calculating
  • Magnification has no units (it's a ratio)
  • 1 mm = 1000 um
  • Use the MAI triangle to rearrange the formula
  • Electron microscopes have higher magnification and resolution

* IGCSE Practice Questions

Question 1 [2 marks]

A cell has an actual size of 0.05 mm. A student draws it with an image size of 25 mm. Calculate the magnification.

* CLICK TO REVEAL ANSWER

Working:

  • Magnification = Image size / Actual size
  • Magnification = 25 / 0.05
  • Magnification = x500
Question 2 [2 marks]

A drawing of a cell is 30 mm wide. The magnification is x600. Calculate the actual size of the cell in micrometres (um).

* CLICK TO REVEAL ANSWER

Working:

  • Actual size = Image size / Magnification
  • Actual size = 30 / 600 = 0.05 mm
  • Convert to um: 0.05 x 1000 = 50 um
Question 3 [1 mark]

A microscope has an eyepiece lens of x10 and an objective lens of x40. What is the total magnification?

* CLICK TO REVEAL ANSWER
  • Total magnification = 10 x 40 = x400
Question 4 [2 marks]

State one advantage of using an electron microscope compared to a light microscope.

* CLICK TO REVEAL ANSWER
  • Higher magnification (1 mark)
  • Higher resolution / can see more detail (1 mark)
